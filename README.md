# Prerequisites
- [Anancoda with Python 3.7 or higher](https://www.anaconda.com/distribution/#download-section)
- [Node JS](https://nodejs.org/en/)
- [Gramex 1.58 or higher](https://github.com/gramener/gramex)

# Setup
- Clone the repo `git clone https://github.com/nikhilkabbin/adverse-events`
- Install JS Dependencies, run `npm install`
- Copy `data` folder to root from [here](https://drive.google.com/drive/folders/14SEH5I_kObNag8i0maKz7yUhzpNvlEHL?usp=sharing)
- Run `gramex`
